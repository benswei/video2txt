🛠️ Python & FFmpeg 国内极速安装修复指引

                
            

            

                
                
                

                    
🐍 1. 安装 Python 环境

                    
本软件核心转录引擎依赖 Python 3.10+ 环境。部分电脑未检测到 Python，请按以下步骤安装：

                    

                        

                            Windows 10/11 国内极速免代理直链：

                            🚀 华为云 Python 3.10.11 一键下载
                        

                        

                            🚨 极其重要注意事项：

                            运行下载的安装包时，必须在第一步勾选最下方的 “[√] Add Python 3.10 to PATH”（将其加入系统环境变量），然后再点击 Install Now 安装！若未勾选，本软件仍将无法识别。
                        

                    

                


                
                

                    
🎵 2. 安装 FFmpeg 音视频工具

                    
FFmpeg 用于视频和音频伴音的提取、格式转换等。若未找到，请按以下步骤配置：

                    

                        

                            第一步：极速下载解压包（国内加速代理）：

                            🚀 FFmpeg Windows 极速解压包下载
                        

                        

                            第二步：解压并配置环境变量（Path）：
                            
    
                                
  1. 下载后，将 `.zip` 压缩包解压到一个固定的英文路径下（例如解压到 D:\\ffmpeg）。
    2. 
                                
  2. 解压后确认存在 D:\\ffmpeg\\bin\\ffmpeg.exe 文件。
    3. 
                                
  3. 在 Windows 搜索框中搜索“环境变量”，打开“编辑系统环境变量” -> 点击“环境变量”按钮。
    4. 
                                
  4. 在“系统变量”列表中找到 Path，双击编辑它，点击“新建”，将 D:\\ffmpeg\\bin 填入并保存退出。
    5. 
                            

                        

                    

                


                

                    配置完成后，双击主界面上方的“环境依赖检测”横幅，即可重新检测状态。
